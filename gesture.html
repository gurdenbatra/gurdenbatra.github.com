Gesture Glove

Ideation, Prototyping, Coding

April 2013

Project to prototype and envision a new kind of gesture based input device. Worked with group of three to create a flex capacitative sensor which was installed inside a glove and then using Arduino it interfaced with Spotify API to control playing music.
